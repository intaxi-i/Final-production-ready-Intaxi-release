9. ВАЖНОЕ ОПЕРАЦИОННОЕ ЗАМЕЧАНИЕ ПО CLOUDFLARE PAGES
Код фронтенда переведён на новый Pages project, но для корректного runtime на app.intaxi.best нужно включить Pages/Workers compatibility flag:
- nodejs_compat
Для Production и Preview.
Без этого Cloudflare Pages может показывать экран:
- Node.JS Compatibility Error
- no nodejs_compat compatibility flag set
Это не проблема исходного кода архива, а настройка Cloudflare Pages project.
