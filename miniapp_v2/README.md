# Intaxi Mini App V2

Mini App V2 is the main product interface for passengers, drivers and admins.

## Rules

- All business actions call Backend V2 API.
- UI must not own pricing, matching, wallet or status transition rules.
- Bot and Mini App must show the same backend state.
- Admin surfaces must use the same admin API.

## First screens

- profile;
- city create;
- city available orders;
- counteroffers;
- current trip;
- driver online/offline;
- driver payment details during trip;
- wallet;
- support;
- rating;
- special mode toggle/theme;
- admin dashboard;
- driver verification;
- payments;
- commission rules;
- orders/trips;
- support tickets;
- audit log.
